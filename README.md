# amoCRM Payments Dashboard

Локальный веб-дашборд для контроля доплат по менеджерам.

## Что нужно

- Node.js 18+ или новее
- `long-lived token` из private integration amoCRM

## Настройка

1. Скопируй `.env.example` в `.env`
2. Заполни:
   - `AMO_BASE_URL`
   - `AMO_ACCESS_TOKEN`
   - `PORT`

Пример:

```env
AMO_BASE_URL=https://pomaho.amocrm.ru
AMO_ACCESS_TOKEN=your_long_lived_token
PORT=3000
```

## Запуск

```powershell
npm.cmd start
```

После запуска открой:

```text
http://127.0.0.1:3000
```

## Что показывает первая версия

- выбор периода
- количество менеджеров
- количество сделок
- общую сумму доплат
- таблицу по ответственным менеджерам

## Правила отбора

- в дашборд попадают только сделки с `Плановой датой доплаты` в выбранном периоде
- `Доплата = Сумма сделки - Сумма предоплаты`
- сделки с доплатой `<= 0` исключаются
- сделки без обязательных полей оплаты исключаются
- стадии `Успешно реализовано`, `Отложено / заморожено`, `Закрыто и не реализовано` исключаются
- сделки на стадии `Отгрузка` остаются в отчете, если доплата еще положительная
